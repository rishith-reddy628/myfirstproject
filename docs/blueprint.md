# **App Name**: EcoCrafter

## Core Features:

- Craft Idea Submission: Allow users to submit their own eco-friendly craft ideas, including title, type of material, video tutorial link, and a description, stored in Firestore.
- Craft Idea Browsing: Enable users to browse craft ideas, filter by material type, and watch embedded YouTube tutorial videos.
- Craft Idea Rating and Review: Allow users to rate and review craft ideas to promote higher quality submissions. Reviews and ratings will be stored in Firestore.
- AI Craft Idea Generator: Generate unique eco-friendly craft ideas based on the type of waste material specified by the user. Uses an LLM tool to help select creative and original ideas.
- Waste Segregation Guide: A dedicated section showing video tutorials on separating dry and wet waste, and providing educational articles on waste management, powered by Firestore.
- User Authentication: Implement email/password-based authentication using Firebase Authentication to manage user accounts and profiles, which are stored in Firestore.
- Community Showcase: Create a community area where users can post images of their crafts and comment on other people's submissions, facilitated by Firestore subcollections for posts and comments.

## Style Guidelines:

- Primary color: Natural green (#8FBC8F), symbolizing sustainability and growth.
- Background color: Soft, desaturated beige (#F5F5DC) providing a neutral backdrop.
- Accent color: Warm brown (#A0522D), for highlights and call-to-action buttons, complementing the natural aesthetic.
- Body and headline font: 'PT Sans' for a modern, humanist feel that is legible and inviting.
- Use line icons that represent sustainability, recycling, and different types of craft materials.  These icons will be simple, modern, and monochrome.
- Design a clean, card-based layout that is intuitive, and responsive across devices.
- Incorporate subtle animations such as fade-ins or transitions when content loads, providing a smooth user experience.
